# NexCart

NexCart is a modern, full-stack e-commerce platform designed to demonstrate real-world software engineering practices and enterprise application architecture.

The project is built using **ASP.NET Core (.NET 10)** for the backend and **React** for the frontend, following a clean multi-layer architecture that emphasizes maintainability, scalability, and separation of concerns.

Rather than being a simple CRUD application, NexCart implements many of the features found in commercial e-commerce systems, including secure authentication, product catalog management, shopping carts, order processing, payment integration, inventory management, discount management, customer reviews, and an administrative dashboard.

The project also showcases modern backend development practices such as CQRS with MediatR, Redis caching, background processing, structured logging, validation pipelines, Docker containerization, automated testing, and CI/CD workflows.

NexCart is intended as a portfolio project that reflects how enterprise-grade .NET applications are designed, developed, tested, and deployed.
